## 🚀 الاستخدام

### للمسؤول (Admin)
```
1. الدخول: admin@test.com / password
2. سيتم التوجيه إلى: /admin/dashboard
3. الوصول لجميع الموارد: Students, Teachers, Courses, etc.
4. عرض الإحصائيات والتقارير
```

### للمستخدم (User)
```
1. التسجيل: إدخال البيانات (role يُعين تلقائياً = 'user')
2. الدخول: user@test.com / password
3. سيتم التوجيه إلى: /user/courses
4. استعراض الدورات والتسجيل فيها
```

## 📊 قاعدة البيانات

### الحقل الجديد
```
Column: role
Type: ENUM('user', 'admin')
Default: 'user'
Nullable: false
```

### أمثلة البيانات
```
| ID | Name  | Email           | Role  |
|----|-------|-----------------|-------|
| 1  | Admin | admin@test.com  | admin |
| 2  | Ahmed | user@test.com   | user  |
| 3  | Sarah | sarah@test.com  | user  |
```

## 🔄 عملية تدفق المستخدم

### عند التسجيل الجديد
```
1. اضغط Register
2. أدخل البيانات
3. يتم إنشاء مستخدم جديد بـ role = 'user'
4. إعادة توجيه لصفحة الدخول
```

### عند الدخول
```
1. أدخل Email و Password
2. التحقق من البيانات
3. فحص الدور:
   - إذا كان admin → /admin/dashboard
   - إذا كان user → /user/courses
```

### عند الخروج
```
1. اضغط على Profile → Logout
2. حذف الجلسة
3. إعادة توجيه للصفحة الرئيسية
```
