## 🚀 الخطوات الجاهزة للتنفيذ

### الخطوة 1: إنشاء مستخدم Admin
```bash
cd c:\xampp\htdocs\studentManagement
php artisan tinker

# ثم اكتب:
User::create([
    'name' => 'Admin',
    'email' => 'admin@test.com',
    'password' => Hash::make('password123'),
    'role' => 'admin'
])

# ثم اضغط Ctrl+D للخروج
```

### الخطوة 2: اختبار الدخول كـ Admin
```
URL: http://localhost/studentManagement
Email: admin@test.com
Password: password123

Expected Result:
✅ تحويل تلقائي إلى /admin/dashboard
✅ عرض لوحة التحكم مع الإحصائيات
✅ الـ Navbar يعرض Admin Links
```

### الخطوة 3: اختبار التسجيل الجديد كـ User
```
اضغط Register
أدخل:
- Name: أحمد محمد
- Email: user@test.com
- Password: password123
- Confirm: password123

Expected Result:
✅ إنشاء مستخدم جديد بـ role = 'user'
✅ تحويل تلقائي إلى /user/courses
✅ عرض الدورات المتاحة
```

### الخطوة 4: اختبار الخروج
```
اضغط على اسم المستخدم ▼ في الأعلى
اضغط Logout

Expected Result:
✅ حذف الجلسة
✅ تحويل إلى صفحة الدخول
✅ رسالة: "Logged out successfully"
```
